### Description

The **Fibonacci numbers**, commonly denoted `F(n)` form a sequence, called the **Fibonacci sequence**, such that each number is the sum of the two preceding ones, starting from `0` and `1`. That is,

> <pre>
> F(0) = 0, F(1) = 1
> F(n) = F(n - 1) + F(n - 2), for n > 1.
> </pre>

Given `n`, calculate `F(n)`.

#### Example 1:

> <pre>
> Input: n = 2
> Output: 1
> </pre>

- F(2) = F(1) + F(0) = 1 + 0 = 1.

#### Example 2:

> <pre>
> Input: n = 3
> Output: 2
> </pre>

- F(3) = F(2) + F(1) = 1 + 1 = 2.

#### Example 3:

> <pre>
> Input: n = 4
> Output: 3
> </pre>

- F(4) = F(3) + F(2) = 2 + 1 = 3.

### Constraints:

- `0 <= n <= 30`

### Solution

```java
class Solution {
    public int fib(int n) {
        if (n < 2) return n;
        int a = 0, b = 1, temp;
        for (int i = 1; i < n; i++) {
            temp = a;
            a = b;
            b += temp;
        }
        return b;
    }
}
```
